# Neon Beat Assault

Electron で動く、1画面完結型の縦スクロールシューティングです。上から降ってくる敵機を捌きながら 30 秒耐え、最後に出現するボスを撃破してください。

## セットアップ

```bash
npm install
npm start
```

## 操作方法

- `←→↑↓` または `WASD`: 自機移動
- `Space`: ショット
- `Shift`: 低速移動（弾避け用）
- `B`: ビートボム発動（弾消し＋広範囲ダメージ）
- `Enter`: タイトル / リザルト画面からゲーム開始

## ゲーム内容

- 通常ステージでは軽量ドローンと重装ドローンがランダム出現します。
- 敵を倒すとスコアとコンボが増加します。
- ボムゲージは時間経過でリチャージされ、100% で再使用できます。
- 30 秒経過でボスが出現し、撃破するとステージクリアです。

## 実装メモ

- Electron の `BrowserWindow` 上で Canvas 描画しています。
- 外部アセット不要で遊べるよう、図形描画だけで演出を構成しています。
